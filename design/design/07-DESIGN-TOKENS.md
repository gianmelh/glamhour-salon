# Glamhour Design Tokens

Extracted from the exported Figma PDF canvases for the Phase 4 component system.

## Visual Direction

- Mobile-first frames are approximately 390px wide.
- Operational screens use a pale lavender canvas with white cards.
- Authentication screens use a white-to-purple vertical gradient.
- Primary actions use a bright-purple to dark-purple vertical gradient.
- Icons are simple, rounded, thin outline icons.

## Core Tokens

| Token | Value | Use |
|---|---|---|
| Ink | `#101827` | Headings and primary copy |
| Muted | `#748096` | Supporting copy |
| Canvas | `#f4f3ff` | App page background |
| Surface | `#ffffff` | Cards, fields, navigation |
| Lavender | `#eee9ff` | Selected and promotional surfaces |
| Primary | `#7c3aed` | Icons, links, active controls |
| Primary dark | `#4c1d95` | Gradient actions |
| Border | `#e1e4ec` | Cards and controls |

## Scale

- Typography: 12px labels, 14px body, 16px card titles, 24px page titles.
- Spacing: 4px base rhythm; 16px page gutter; 24px section gap.
- Controls: 48px standard height.
- Radius: 12px controls, 16px cards, 20px feature cards, 24px sheets.
- Shadows: subtle card elevation; stronger purple action shadow.
- Icons: Lucide outline icons at 18–22px with approximately 1.8px strokes.
